# MNIST Machine Learning Classification

This repository demonstrates the implementation and comparison of multiple
machine learning algorithms on the MNIST handwritten digit dataset using
a collaborative GitHub workflow.

## Project Objectives
- Implement machine learning algorithms on MNIST
- Improve accuracy through algorithmic changes
- Practice GitHub collaboration using pull requests

## Algorithms Implemented
- Decision Tree Classifier (~90% accuracy)
- K-Nearest Neighbors (~97% accuracy)
- Support Vector Machine (~98% accuracy)

- `src/` : Source code for ML models
